Methods of Machine Learning Employed

Health Matters includes the implementation of the KNN Algorithm. The KNN Algorithm is one of the most common classification algorithms was implemented using a database in which the data points were separated into several classes to predict the classification of a new sample point. Some accuracy metrics we achieved include an ROC AUC of 0.72 and PR AUC = 0.41.

Our team also used Random Forests, a flexible machine learning model was implemented for classification.
Some accuracy metrics we achieved include an ROC AUC of 0.84 and a PR AUC of 0.63.

Our hack also uses Logistic Regression, a statistical model was used to analyze the dataset carefully and perform binary classification.
Some accuracy metrics we achieved include an ROC AUC of 0.68 and PR AUC of 0.29.
