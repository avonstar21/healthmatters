WHAT, WHY, HOW

“Health Matters” provides a user with personalized preventative measures based on their age, sex, and/or pregnancy status. Our tool supplies users and doctors with pertinent factors connected to common preventative diseases.

According to the CDC, preventative care can save over 100,000 lives each year. Additionally, lifestyles can be improved and treatments work best when illnesses are detected earlier. Using our hack “Health Matters”, users can obtain personalized recommendations about clinical preventative measures tailored to his/her age, sex, and/or pregnancy status.

Through the power of analytics and machine learning (ML), patient data with common preventable disease such as diabetes and heart disease can be better understood. Implementation of ML algorithms such as knn, linear regression, random forest, and lasso regression can provide value clinical insights that can aid users towards healthier lifestyles.

Technologies used

The following are the technologies we used to implement our project, what they are and how we used them in our project.

Healthfinder API

The team at Health Matters used the HealthFinder API, which provides a list of possible diseases to look out for on the basis of age and gender. We used this API to display a list of possible diseases for the user.

ML Algorithms

Machine Learning algorithms including logical regression, random forest and k nearest neighbors are utilized in our Implementation. The team used these methods to train the dataset provided by the HealthFinder API and come up with results that indicate the accuracy of diabetes diagnosis. Health Matters predicts key important factors of diabetes.

Python Packages

The team used the NumPy, Pandas, matplotlib, scikit-learn libraries to implement ML algorithms and visualize the opioid prescription rates.

Front-end Technologies and JS

HTML/CSS/Bootstrap was used throughout the project as the ‘front end’ of the website where we displayed the outputs of machine learning and helped users further diagnose themselves. Javascript was used for adding actions to our HTML pages and making them into actual websites.
